辦理活動

積極辦理獨立遊戲開發的相關活動

異校遊戲大亂鬥(SSG)

想要完成一款出色的遊戲， 都需要經歷「白紙測試」的考驗。 勇敢地將開發中的遊戲呈現給玩家試玩， 透過不斷的回饋與改進提升遊戲品質。

活動報名

Global Game Jam(GGJ)

Game Jam 是一個開發遊戲的有趣活動，參加者要在短時間內和其他陌生的遊戲開發者合作，根據特定主題製作出一款遊戲。

Faust Game Jam(FGJ)

原為與日本一起辦理的 Fukushima Game Jam，後為了紀念 Faust Li（李家屹）為台灣遊戲社群的貢獻，邀請各位在台灣的遊戲開發社群參與者，一起來紀念他。本活動仍以「FGJ」來紀念 Faust。

因為沒有固定聚會場所，目前僅能固定辦理GGJ、FGJ、SSG(異校遊戲大亂鬥)

請問活動需要費用嗎？

這個社群本身就是非營利組織，目前並無營運之考量。所有活動目前皆無收費。如果你真的認可我們，歡迎抖內我們！
